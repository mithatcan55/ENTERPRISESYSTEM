# Frontend Starter Kit

Bu katmanin amaci yeni bir modulu sisteme en az kod ile takabilmektir.

Hedef:
- route kaydi
- menu kaydi
- page header
- standard data table
- standard form
- permission/tcode guard
- localization namespace

Yeni bir CRUD modulu acarken ilk referans dosya:
- `src/starter-kit/contracts/crud-module.ts`

Beklenen yol:
1. module contract tanimla
2. route ve menu kaydini ekle
3. standard list/form ekranina schema ver
4. backend endpointlerini bagla
